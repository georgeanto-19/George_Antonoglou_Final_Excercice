DB
Step 1- Χρειάζεται να βρούμε το URL της βασης MongDB που χρησιμοποιούμε στο αρχείο BackEnd/dev.env στο πεδίο Mongo_URI
BACKEND
Step 2 - Ανοίγουμε Terminal στο FilePath BackEnd/src και τρέχουμε τις εντολές  npm install , 
Step 3 - μετα τρέχουμε  npm install mongoose dotenv, npm install bcrypt jsonwebtoken , npm run dev
Step 4 - Καλούμε απο Postman  το enpoin Health για να δουμε οτι λειτουργεί σωστά η εφαρμογή μας
Step 5 - Καλούμε το enpoint 'http://localhost:4000/api/initialDb' για να δημιουργήσει τα initial data της εφαρμογής μας 
FRONTEND 
Step 6 - Ανοιγουμε Terminal στο FIlePath FrontEnd/src  και τρέχουμε τις εντολές Npm install
Step 7 - μετα τρέχουμε npm start
Step 8 - Αφου τρέξει επιτύχως μπορούμε να δουμε το FrontEnd app στο url http://localhost:4200/, μπορούμε να κανουμε Login με username: admin & pass: admin και να πλοηγηθουμε στην υπολοιπη εφαρμογη.
